<h1>Smart-Weather-App</h1>

<p>This app checks the current weather conditions and determines if it is an inclement weather.</p>
<p>If it is, the user is sent an alert to close the opened windows.</p>
<p>The user is also notified if weather is forecasted to be bad.</p>
